# samp-rust-example


An example usage of [samp sdk](https://github.com/ZOTTCE/samp-sdk) by [ZOTTCE](https://github.com/ZOTTCE) that exports a single function

```
rust_say();
```
With no paramters, 
It should output "Hello World!" to the console.


Pawn example code:
```sourcepawn
#include <a_samp>

native rust_say();

main() {
	rust_say();
}
```
